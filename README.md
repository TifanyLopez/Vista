# Instalacion de archivos requeridos
Este es un proyecto de aplicación web en Flask para gestionar información sobre aves.

## Crear el entorno virtual
python -m venv venv

## Crear el archivo
pip freeze > requerimientos.txt

 **Intalarlo**
pip install -r requerimiento.txt

## Recurso
python 3.13.5

microframework web
flask 

  gestor de base de datos:
   MySql
